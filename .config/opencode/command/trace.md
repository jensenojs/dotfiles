# 内核追踪分析

协助进行内核级别的追踪和调试:

## 追踪工具

1. **Ftrace**
   - 函数追踪和事件追踪
   - 分析内核执行路径
   - 设置 tracepoint 和 kprobe

2. **BPF/eBPF**
   - 编写 bpftrace 脚本
   - 内核态和用户态数据收集
   - 实时监控和统计

3. **SystemTap**
   - 复杂的动态追踪脚本
   - 内核函数插桩
   - 数据聚合和分析

## 分析场景

- **系统调用追踪**: 分析应用与内核的交互
- **中断和调度**: 追踪中断处理和进程调度
- **内存管理**: 页面分配、回收和缺页异常
- **文件系统**: I/O 路径和缓存行为
- **网络栈**: 数据包处理和协议栈分析

## 输出分析

- 解释追踪日志和时序关系
- 识别性能异常和瓶颈
- 提供调优建议和解决方案

请描述要追踪的内核行为或提供现有的追踪数据。
